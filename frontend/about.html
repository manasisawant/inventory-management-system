About the Project

This project is a web-based Inventory Management System developed as a final semester group project for the BSc Computer Science program.

What is this project?

The Inventory Management System helps businesses manage products, track stock levels, and control access through role-based authentication. It is designed to simplify inventory operations and improve efficiency.

How are we building it?

The system is being developed using a structured approach where frontend, backend, and database components are handled separately and later integrated into a complete solution.

  • Frontend: HTML, CSS, Bootstrap
  • Backend: Java
  • Database: PostgreSQL
  • Version Control: Git & GitHub

Team Collaboration

This is a group project developed by three members. Each member is responsible for a specific part of the system, allowing parallel development and better collaboration using GitHub.
